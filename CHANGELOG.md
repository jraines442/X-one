# Changelog

All notable changes to X-one will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-01-XX

### Added

- Initial release of X-one
- CLI tool for scaffolding modern web projects
- Support for React (Vite) template
- Support for Next.js template
- Support for Vite vanilla TypeScript template
- `x-one create` command for project generation
- `x-one dev` command for development server
- `x-one build` command for production builds
- Zero-configuration approach with sensible defaults
- TypeScript support in all templates
- Git initialization for new projects
- Comprehensive documentation
- Contributing guidelines
- MIT License
